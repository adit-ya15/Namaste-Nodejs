## Overview
This lecture serves as an introduction to the course and sets the mindset required to become a strong backend developer.

## Key Learning
- Curiosity is a **core skill** for developers.
- A good developer does not just use tools, but asks:
  - *Why does this work?*
  - *How does this work internally?*
- Being curious leads to deeper understanding and better problem-solving.

## Takeaway
> Curiosity separates an average developer from a great one.  
> Always question how things work under the hood.
